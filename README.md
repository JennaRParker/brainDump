# Brain Dump

### _A simple, mobile platform for users to anonymously share their thoughts and comment on posts._

## Technologies Used

* _Python_
* _Django_
* _PostgreSQL_
* _Heroku_
* _CSS_
* _HTML_
* _JavaScript_
